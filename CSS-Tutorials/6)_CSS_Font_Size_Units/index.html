CSS Font Size Units

Understanding absolute, relative, and viewport-based units

1. Pixels (px)

Absolute unit. Fixed size regardless of parent or root.

This text is 20px.

2. Em (em)

Relative to the font size of the parent element.

Parent is 16px → This text is 2em (32px).

3. Rem (rem)

Relative to the root (html) font size, usually 16px by default.

This text is 1.5rem (24px if root is 16px).

4. Percent (%)

Relative to the parent element’s font size.

Parent is 20px → This text is 120% (24px).

5. Viewport Units (vw, vh)

Relative to the size of the viewport.

This text is 5vw (5% of viewport width).
This text is 5vh (5% of viewport height).
