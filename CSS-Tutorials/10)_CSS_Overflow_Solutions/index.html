CSS Overflow Properties

Controlling content that exceeds its container

1. overflow: visible (default)

Content spills outside the box if it’s larger than the container.

This text is very long and will overflow outside the box without clipping or scrollbars.

2. overflow: hidden

Extra content is clipped and not visible. No scrollbars are provided.

This text is very long and will be clipped. You won’t see the overflow.

3. overflow: scroll

Always shows scrollbars, even if content fits.

This text is very long and will require scrolling. Scrollbars are always visible.

4. overflow: auto

Scrollbars appear only when needed (if content overflows).

This text is very long and will require scrolling. Scrollbars appear only when necessary.

5. overflow-x and overflow-y

You can control horizontal and vertical overflow separately.

This text is very long and will require vertical scrolling. Horizontal overflow is hidden.
