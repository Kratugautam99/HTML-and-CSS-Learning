CSS Position Property

Understanding static, relative, absolute, sticky, and fixed positioning

1. Static (default)

Elements are positioned according to the normal document flow. top, left, etc. have no effect.

Static Box

2. Relative

Positioned relative to its normal position. Space is still reserved in the layout.

Relative Box (moved 20px)

3. Absolute

Positioned relative to the nearest positioned ancestor (not static). Removed from normal flow.

Parent (relative)
Absolute Box

4. Sticky

Behaves like relative until a scroll threshold is reached, then sticks to the defined edge.

Sticky Box (sticks to top)

Scroll down to see sticky behavior.

Extra content to enable scrolling...

5. Fixed

Positioned relative to the viewport. Stays in place even when scrolling.

Fixed Box
